# Autodidact Releases

This repository hosts the release builds for [Autodidact](https://github.com/matthewmusson/Autodidact).

## Download

Visit the [Releases](https://github.com/matthewmusson/Autodidact-Releases/releases) page to download the latest version.

## System Requirements

- macOS (Universal binary supports both Apple Silicon and Intel)

## Installation

1. Download the DMG file from the latest release
2. Open the DMG
3. Drag Autodidact to your Applications folder

*Note: This is an automated release repository.
